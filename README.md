# ml-digits-project
Simple implementation of 7 ways of recognizing handwritten digits for scikit-learn digits, MNIST, EMNIST digits.
The single layer Perceptron/Adaline/Logistic Regression and the MLP are implemented from scratch whereas the second MLP_pt and CNN are made using Pytorch. 
The Excel file shows the comparison between the models with respective epochs, fixed learning rate, training accuracy and testing accuracy to check for overfitting. 
The plots show how different models see the weights, and the loss function and accuracy of the neural networks.
